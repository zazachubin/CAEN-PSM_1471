Installation Instructions
-------------------------

Please follow the installation instructions in order to correctly install the 
PSM LabView Instrument Driver into your environment.

1) copy the "CAEN PSM" folder into your current LabView release path 
   e.g. for LabView 2009 "...\National Instruments\LabVIEW 2009\instr.lib"
2) Run LabView
3) the CAEN PSM instrument driver should appear on you "Instrument I/O" -> "Instr Drivers" palette.

Further information or support on: http://www.caen.it/csite/ProductSupport.jsp
